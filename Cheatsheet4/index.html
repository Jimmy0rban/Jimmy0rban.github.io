Sample Questions - Basic Web Techniques - Advanced CSS

Box Model

1. Block elements have which kind of characteristics?

  • Always start on a new line.
  • Take up as much horizontal space as possible (the full width of the container or browser window if there is no container)
  • Will respect width and height CSS properties.
  • Horizontal and vertical margins both work.

2. Inline elements have which kind of characteristics?

  • Do not start on a new line
  • Only use as much horizontal space as required by the content
  • Do not accept width and height CSS properties
  • Margins will work horizontally, but not vertically
  • Padding works on all sides, but the top and bottom may overlap other elements.

3. What is the margin of an HTML element? Where can one find it?

  • Margin is the space outside the element's border.
  • It creates space between the element and other elements around it.
  • Margins collapse in certain situations (e.g., vertical margins of adjacent block elements), meaning overlapping margins may merge into a single margin.

4. What is the padding of an HTML element? How does it differ from margin?

  • Padding is the space inside the element’s border, between the content and the border.
  • It increases the space between the content of the element (like text) and the edge of the element itself.
  • Padding doesn’t collapse, and it pushes the border outward as it increases.

Classes and IDs

1. Write down a selector for a certain class:

.my-class { /* CSS properties go here */ }

2. Write down a selector for a certain HTML element of a certain class:

element.class-name { /* CSS properties go here */ }

3. Write down a selector for a certain id:

#my-id { /* CSS properties go here */ }

4. Write down a selector for a certain HTML element of a certain id:

element#id-name { /* CSS properties go here */ }

Floating

1. CSS declaration to make an element float to the left:

element { float: left; }

2. Make an element stop floating (for specific or all directions):

.element { clear: left; /* or clear: both; */ }
