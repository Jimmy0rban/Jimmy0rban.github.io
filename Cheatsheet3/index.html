Cheat sheet:

Basic Web Techniques - Lists and Tables

Html for Lists

  1. Start tags for
    • an ordered list - Answer: <ol>
    • an unordered list - Answer: <ul>
    • a description list - Answer: <dl>
    • a list item - Answer: <li>
    • a description term - Answer: <dt>
    • a description data - Answer: <dd>

Css for Lists

  1. Make bullet a square 
    <ul style="list-style-type: square;">
    <li>Item 1</li>
</ul>
  2. Make bullet an image named "my-bullet.png" 
    <ul style="list-style-image: url('path');">
    <li>Item 1</li>
</ul>
  3. Make bullet an uppercase roman number 
    <style>
    ul.roman {
        list-style-type: upper-roman;
    }
</style>
<ul class="roman">
    <li>Item 1</li>
</ul>
  4. Set bullet position to be inside the list item's text flow 
    <style>
    ul {
        list-style-position: inside; 
    }
</style>
<ul>
    <li>First item with inside bullet</li>
    <li>Second item with inside bullet</li>
    <li>Third item with inside bullet</li>
</ul>
  5. Set bullet position to be outside the list item's text flow 
    <style>
    ul {
        list-style-position: outside; 
    }
</style>
<ul>
    <li>First item with outside bullet</li>
    <li>Second item with outside bullet</li>
    <li>Third item with outside bullet</li>
</ul>

Html for Tables

  1. Start tags for
    • a table - Answer: <table>
    • a table row - Answer: <tr>
    • a table header - Answer: <th>
    • a table header cell - Answer: <th> Headline </th>
    • a table body - Answer: <tbody>
    • a table cell - Answer: <td>
    • a table footer - Answer: <tfoot>
    • a table footer cell - Answer: <tfoot> <td> footer cell </td> </tfoot>

Css for Tables

  1. Black dotted border of 1 px width. 
    .dotted-border {
    border: 1px dotted black;
}
  2. Solid border of 1 px width having a red color and rounded corners 
    .border {
    border: 1px solid red;
    border-radius: 10px;
}
  3. Merge the borders of table cells so that they appear as one line 
    .border {
    border-collapse: collapse;
    border: 1px solid black;
}
